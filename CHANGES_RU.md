# Что изменено в этой сборке

1. Исправлен разбор красиво отформатированного Apple TTML: пробельные XML-узлы между timed `<span>` больше не превращаются в слова с таймкодом `0–0`.
2. Упреждение анимации подлёта следующего слова изменено с `400 мс` на `0 мс` во всех runtime-вариантах AMLL Core 0.4.0:
   - `vendor/applemusic-like-lyrics-core/dist/amll-core.mjs`
   - `vendor/applemusic-like-lyrics-core/dist/amll-core.cjs`
3. Vite настроен использовать локальную исправленную копию AMLL Core из папки `vendor`, поэтому GitHub Actions не затрёт изменение при установке зависимостей.
4. Добавлен workflow `.github/workflows/deploy-pages.yml` для автоматической сборки и публикации сайта через GitHub Pages.

При нулевом упреждении используется выражение `wordDe - 0.0`, то есть подлёт начинается без прежних дополнительных 400 мс заранее.
